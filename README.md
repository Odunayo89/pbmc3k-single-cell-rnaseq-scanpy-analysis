# Single-Cell RNA-seq Analysis of PBMC3k Dataset Using Scanpy

This project demonstrates a complete single-cell RNA sequencing (scRNA-seq) analysis workflow using the PBMC 3K dataset from 10x Genomics. The analysis is performed in Python using the Scanpy framework and follows a standard pipeline used in many single-cell transcriptomics studies.

The goal of this project is to identify transcriptionally distinct immune cell populations and characterize their marker genes.

---

## Dataset

PBMC3k dataset from a healthy donor generated using the 10x Genomics Chromium platform.

Dataset source:
https://support.10xgenomics.com/single-cell-gene-expression/datasets/1.1.0/pbmc3k

Cells analyzed: ~2,700  
Genes detected: ~13,000  

Peripheral Blood Mononuclear Cells (PBMCs) include multiple immune cell types such as:

- T cells
- B cells
- Natural Killer (NK) cells
- Monocytes
- Dendritic cells

## Analysis Workflow

The analysis follows a standard single-cell RNA-seq pipeline:

1. Load gene-cell count matrix generated by Cell Ranger
2. Quality control filtering of cells and genes
3. Library size normalization
4. Log-transformation of expression values
5. Identification of highly variable genes
6. Dimensionality reduction using PCA
7. Construction of a nearest-neighbor graph
8. Cell clustering using the Leiden algorithm
9. Visualization using UMAP
10. Identification of cluster-specific marker genes
11. Biological interpretation of clusters using canonical immune cell markers

## Key Visualizations

### UMAP Clustering
UMAP projection of PBMC cells showing transcriptionally distinct clusters identified using the Leiden algorithm.

### Marker Gene Expression
Expression patterns of canonical immune markers:

- MS4A1 → B cells
- LYZ → Monocytes
- NKG7 → NK cells / cytotoxic T cells

### Dot Plot
Visualization of marker gene expression across clusters showing both:

- Mean gene expression
- Fraction of cells expressing the gene

### Differential Gene Expression
Cluster-specific marker genes identified using the Wilcoxon rank-sum test.

## Biological Interpretation

Clusters correspond to major PBMC immune cell populations. Marker genes help identify cluster identities:

| Marker Gene | Cell Type |
|-------------|-----------|
| MS4A1 | B cells |
| LYZ | Monocytes |
| NKG7 | NK / Cytotoxic T cells |

These marker patterns align with expected immune cell composition in PBMC datasets.

## Example Code Snippet

```python
import scanpy as sc

adata = sc.read_10x_mtx("hg19", var_names="gene_symbols")

sc.pp.normalize_total(adata)
sc.pp.log1p(adata)

sc.pp.highly_variable_genes(adata)
adata = adata[:, adata.var.highly_variable]

sc.tl.pca(adata)
sc.pp.neighbors(adata)
sc.tl.leiden(adata)

sc.tl.umap(adata)

sc.pl.umap(adata, color="leiden")
